    What Tanzu Platform adds

    Enterprise operations, fleet-wide security, AI-ready services, and multi-foundation governance — built on Elastic Application Runtime.

    Tanzu 10.4 · Security
    🛡️
    Fleet-wide CVE patching
    AI compresses CVE-to-exploit from weeks to hours. Tanzu Hub pinpoints exactly which buildpack version affects which apps. One click generates a compatibility-safe Upgrade Plan. Developers click "Restage" — no code changes. Customers run 100,000+ automated patches monthly.
    Tanzu 10.3+ · AI
    🤖
    MCP Server Marketplace
    Publish MCP servers as CF services. Teams discover and bind with the standard CF CLI — credentials injected via VCAP_SERVICES. Off by default; admin approval per org prevents shadow AI. Spring Cloud Gateway provides automatic network isolation.
    Tanzu 10.4 · AI
    ⚡
    Agentic AI on CF
    GenAI tile adds GPU-hosted LLM endpoints to the marketplace. cf create-service ai-models chat-and-tools-model — credentials auto-injected. Spring AI + MCP enables server-driven agentic loops. HATEOAS constrains agent behaviour safely at each step.
    Tanzu Spring · Security
    🔒
    Spring supply chain security
    SLSA Level 3-validated Java dependencies — 100,000+ validated builds. CVE-only patch releases: apply security fixes without version upgrades. Spring Application Advisor delivers a 70% reduction in upgrade time. Spring Boot 3.5 supported through 2032.
    Ops Manager
    🏗️
    Tile-based platform operations
    Install, configure, and upgrade EAR, data services, and security tiles through a UI. Handles certificate rotation, stemcell patching, and VM upgrades. 20+ tiles: Compliance Scanner, CredHub, Harbor, Healthwatch, Concourse, Cloud Service Brokers.
    Tanzu Hub
    🌎
    Multi-foundation management
    Centralised dashboard for multiple CF foundations. Fleet-wide vulnerability posture across buildpacks, stemcells, stacks, and tiles. Portfolio assessment and modernisation. Works in air-gapped environments. Loggregator Firehose nozzle to Datadog, Splunk, or Prometheus.
